C++ Platformer

Unity Rescue Scenario

This is a project created in Unity using a model I created in Maya. You control a flying train which can then use the claw on the back in order to pickup boulders from a landslide. The landslide is created using a spawner and particle effects.

Python Boss-Beaters

This is a group project where we used Python and a framework called PyASGE to create a top-down, turn-based game. The player is able to use both ranged and melee attacks. They need to clear through several maps which each have different enemies and bosses.
